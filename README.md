# earthquake-magnitude-predictor-
# 🌍 Earthquake Magnitude Detector

## 📌 Overview
This project predicts the magnitude of earthquakes using machine learning techniques.  
It analyzes seismic data such as depth, location, and other geological features.

## 🚀 Features
- Data preprocessing and cleaning  
- Machine learning model for magnitude prediction  
- Visualization of earthquake data  
- Easy-to-understand notebook implementation  

## 🛠 Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  

## 📂 Project Structure
- earthquake_magnitude_detector.ipynb → Main project notebook  

## ▶️ How to Run
1. Open the notebook in Jupyter or Google Colab  
2. Run all cells step-by-step  
3. View predictions and results  

## 📊 Output
- Predicts earthquake magnitude based on input features  
- Displays model performance and insights
